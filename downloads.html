LSPDFR Plugins

LSPD First Response (LSPDFR) is the police modification for the PC version of Grand Theft Auto V which completely transforms the game into a law enforcement sim.

UnitedCallouts

UnitedCallouts is adding 20 callouts for the GTA5 LSPDFR modification. It contains different dialogs, different endings and possiblities, version checker, changeable hotkeys.


More information

BetterInteraction

BetterInteraction is an alternative menu to STP and UB with currently some general features, which are listed below for more information. I made this menu, since STP and UB are blocking many features and animations that are in LSPDFR.

More information

Simple Police Helper

The Simple Police Helper (SPH) is a simple menu with features to make the patrols easier. The menu is adding some options for the player, vehicle, time and weather, interactions, animations/scenarios and much more.

Developer Examples

I created a few examples for new developers, who would like to get started.

ExampleCallouts

ExampleCallouts is an open source project and will remain so. This project is there for you to see how callouts of this kind are constructed. It contains a version checker, different dialogs, 4 callouts, different endings and possiblities.

More information

Example Discord Bot

Example Discord Bot is a basic discord bot with some features, commands and permissions. The bot contains currently 6 commands like responding commands, purge command, ban command, game status for the bot and much more.
