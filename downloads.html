LSPDFR Plugins

LSPD First Response (LSPDFR) is the police modification for the PC version of Grand Theft Auto V which completely transforms the game into a law enforcement sim.

UnitedCallouts

UnitedCallouts adds over 19+ callouts for LSPDFR, a popular GTA V modification. Featuring a version checker, custom settings and various scenarios, UnitedCallouts adds plenty of variety to your LSPDFR. The full source code is on GitHub.

BetterInteraction

BetterInteraction is an system used to interact with suspects, persons of interest and pedestrians. Features include ID checks, drug/alcohol testing, calling services, changeable hotkeys and settings, pedestrian transport and much more.

Simple Police Helper

Simple Police Helper (SPH) The Simple Police Helper is a simple menu with features to make the patrols easier. The menu allows you to change time, weather, vehicles and load custom scenarios. This plugin is currently not open source on GitHub.

Developer Examples

I created some examples development content for new developers in case they need a platform to start from. Read the GitHub page for more information about the dependencies you need.

ExampleCallouts

ExampleCallouts is project, which is an example of how to build a basic callout plugin. It contains a version checker, interaction options and 4 callouts with different endings. ExampleCallouts is an open source project and will remain so.
