Troubleshooting

UnitedCallouts


I can't get inside a building, while I am playing a callout like "Burglary into an apartment". How can I fix that?
If you can't get inside a building, you need to install "OpenAllInteriors", which is linked in the UC.ini.
If you don't want to use it, set the callouts for OAI to "false". (They are set to false by default)



I am using CalloutManager and I can't start a callout like "Burglary into an apartment". How can I fix that?
If you can't start the callout you'll have to check the UnitedCallouts.ini. The callout might be set to "false".
The CalloutManager is showing the callout in the game, but you can't play it until the callout is set to "true".



Assembly UnitedCallouts.dll (UnitedCallouts, Version=1.5.7.3, Culture=neutral, PublicKeyToken=null) does not have the required Plugin attribute. How can I fix that?
If you have this error in your log you installed UnitedCallout wrong. Make sure, that UnitedCallouts is in "GTA5/plugins/LSPDFR".
If you don't have a "LSPDFR" folder, create one.

Simple Police Helper

The menu is not working. What now?
The most problem is, that you didn't install NativeUI correctly.
Make sure, that the file is in GTA5/scripts and not in the main directory of GTA5.



How do I open the Simple Police Helper menu?
You can open the menu with F5 by default.
If you want, you can also change the key for your liking in the SPH.ini.
